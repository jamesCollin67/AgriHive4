# AgriHive Design System

## Color Theme
| Role | Hex | Usage |
|------|-----|-------|
| Primary (Honey Gold) | #F4B400 | Buttons, accents, highlights |
| Secondary (Dark Brown) | #4E342E | Text, headers |
| Background (Soft Cream) | #FFF8E1 | App background |
| Accent (Leaf Green) | #66BB6A | Success, healthy status |
| Error (Soft Red) | #EF5350 | Alerts, errors |

## Typography
- **Titles**: Bold (sans-serif-black), 20–28sp
- **Labels**: Medium (sans-serif-medium), 14–16sp  
- **Body**: Regular (sans-serif), 16sp
- **Captions**: Regular, 12sp

## Spacing & Layout
- Card padding: 20dp
- Screen padding: 16–24dp
- Corner radius: 12–16dp
- Card elevation: 4dp (8dp on hover)

## Status Colors (Hive Cards)
- **Green**: Normal range
- **Yellow**: Warning
- **Red**: Critical
